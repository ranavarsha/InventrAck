InventrAck is a PHP & MySQL based inventory management system that allows users to manage products, stock levels, and view inventory analytics.

## Features
- Login & Logout (Session based)
- Add, Edit, Delete Products (CRUD)
- Stock status (Low / High)
- Search functionality
- Dashboard with statistics & chart
- Responsive UI

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript (Chart.js)

## Setup (Local)
1. Install XAMPP
2. Place project in `htdocs`
3. Create MySQL database
4. Import SQL file
5. Run `http://localhost/InventrAck/`
